# BrightTv

   ## Project Overview

This project analyzes user behavior and viewership patterns for BrightTV to support business decisions aimed at growing the subscription base.

The analysis combines user profile data and viewership session data to uncover insights on:

User engagement trends
Content consumption behavior
Factors influencing viewership
Opportunities to increase usage and subscriptions

 ## Business Objective

The CEO of BrightTV aims to increase the subscription base for the current financial year.
This project provides insights to assist the Customer Value Management (CVM) team in achieving this goal.

  ## Data Preparation

  I performed LEFT JOIN between:

viewership (A)
user_profiles (B)

## Key Insights

Certain days of the week show higher engagement
A small number of channels drive most of the consumption
User activity varies significantly by province and demographics
Repeated records indicate multiple sessions per user
